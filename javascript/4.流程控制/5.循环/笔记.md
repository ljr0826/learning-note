# 循环

重复的运行一段代码

js支持三种循环结构（目前就js5而言）：while循环、do while循环、for循环

## while循环

```js
while(条件){
    代码块(循环体);
}
```

```mermaid
graph TD
st((开始))-->if{条件}
if--true-->代码块-->if
if--false-->ed((结束))
```

> 死循环：条件永远满足，永远无法退出循环

## do-while循环

```js
do{
    循环体
}while(条件);
```

```mermaid
graph TD
st((开始))-->代码块-->if{条件}
if--true-->代码块
if--false-->ed((结束))
```

## for循环

```js
for(初始化表达式;条件;条件改变表达式){
    循环体;
}
```

```mermaid
graph TD
start((开始))-->初始化表达式-->if{条件判断}
if--true-->循环体-->条件改变表达式-->if
if--false-->ed((结束))
```

## 循环中的关键字

循环控制语句

- break：跳出循环
- continue：停止当前循环体，进入下一次循环
