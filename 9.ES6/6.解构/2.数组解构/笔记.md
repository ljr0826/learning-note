# 数组解构

数组的本质其实也是对象，[]表示解构数组

```js
const numbers = ["a", "b", "c", "d", [1, 2, 3, 4]];
// const { 0: n1, 3: n2 } = numbers;
// let n1, n2;
// [n1, n2] = numbers;
const [n1, n2, , n4, n5 = 123] = numbers; //[]表示解构数组
console.log(n1, n2, n4, n5);


const numbers = ["a", "b", "c", "d", [1, 2, 3, 4]];
//得到numbers下标为4的数组中的下标为2的数据，放到变量n中
const [, , , , [, , n]] = numbers;
console.log(n); //3

const numbers = ["a", "b", "c", "d", { a: 1, b: 2 }];
//得到numbers下标为4的对象中的属性a，赋值给变量A
const [, , , , { a: A }] = numbers;
// const { a: A } = number[4];//这样也可以
console.log(A);
```

## 补充：剩余项的收集

解构剩余项，使用展开运算符。（收集剩余的信息）

```js
const user = {
  name: "kevin",
  age: 11,
  sex: "男",
  address: {
    province: "四川",
    city: "成都",
  },
};
//解构出name，然后将剩余的所有属性放到一个新对象中，变量名为obj
//name:kevin
//obj:{age:11,sex:"男",address:{...}}
const { name, ...obj } = user; //展开运算符在此处起到一个收集剩余信息的作用
console.log(name, obj);
//数组解构同

const numbers = [324, 7, 23, 0, 5, 53];
//得到数组前两项，分别放到变量a和b中，然后剩余的所有数据放到数组nums中
const [a, b, ...nums] = numbers; //剩余项的解构必须是最后一个，且只能有一个
console.log(a, b, nums);
```

### demo、实现两个变量值的交换

实现交换两个变量的方法

```js
let a = 1,b = 2;
[b, a] = [a, b]; //将两个变量置于一个数组中，对数组进行解构，从而达到交换变量值的目的
console.log(a, b);
```
