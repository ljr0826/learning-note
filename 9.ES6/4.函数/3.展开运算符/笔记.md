# 展开运算符

使用方式：`...要展开的数组或对象`（写法和剩余参数一样，但是使用场景不同）

对今后克隆、react的使用大有用处

```js
//函数求和
function sum(...args) {
  let sum = 0;
  args.forEach((item) => {
    sum += item;
  });
  return sum;
}
//获取一个指定长度的随机数数组
function getRandomNumbers(length) {
  const arr = [];
  for (let i = 0; i < length; i++) {
    arr.push(Math.random());
  }
  return arr;
}
const numbers = getRandomNumbers(10); //获取一个长度为10的随机数数组。对这个数组求和
console.log(sum.apply(null, numbers));
console.log(sum(1, 2, ...numbers, 1, 2)); //产开运算符前后还可以继续写参数
//希望将数组的每一项展开，依次作为参数传递，而不是把整个数组作为一个参数传递----->展开运算符
```

## 对数组展开 ES6

可以应用到很多场景

```js
const arr1 = [1, 3, 43, 5, 435];
//克隆arr1到数组arr2中
const arr2 = [1, ...arr1, 2]; //前后可以随意添加需要的数据
console.log(arr2, arr2 === arr1); // ... false
```

## 对对象展开 ES7

```js
const obj1 = {
  name: "顾昀",
  age: 27,
  love: "长庚",
  address: {
    country: "大梁",
    province: "燕回",
    city: "小镇",
  },
};
//克隆一个对象到obj2
const obj2 = { ...obj1, name: "大胆", address: { ...obj1.address } };
//将对象全部展开就相当于一对对的键值对，重复属性能起到一个覆盖的作用
//原来对象不可变，产生一个新的对象
console.log(obj2);
console.log(obj2.address === obj1.address); //false
```
