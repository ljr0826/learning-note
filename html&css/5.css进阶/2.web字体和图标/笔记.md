# web字体

解决用户电脑上没有安装相应字体，强制让用户下载改字体

使用@font-face指令制作一个新字体

！下载需时间，影响效率因此并不常见

## 字体图标

iconfont.cn  阿里巴巴做的一个字体图标库

应用上分在线字体图标和离线字体图标

1. 在线字体图标

font class生成类样式，复制类名做属性值

unicode复制代码到css内部样式表中，制作一个新字体，复制html实体字符

> 更建议用font class类样式，因为字体图标更应该是样式而非html内容

2. 离线字体图标

下载至本地即可

> 建议开发时用在线字体图标，上线时用离线

习惯上常用i元素(块盒)当图标
